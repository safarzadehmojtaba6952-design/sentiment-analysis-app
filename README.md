# 🧠 پروژه‌ی تشخیص احساسات پیامک

یک برنامه‌ی موبایل ساده با پایتون که احساسات (شاد/غمگین) یک جمله را تشخیص داده و در دیتابیس ذخیره می‌کند.

## ✨ قابلیت‌ها
- تشخیص احساسات با استفاده از هوش مصنوعی (Naive Bayes)
- ذخیره‌سازی در SQLite
- نمایش تاریخچه‌ی پیام‌ها
- رابط کاربری ساده با Kivy

## 🛠️ تکنولوژی‌ها
- Python 3.8+
- Scikit-learn
- SQLite
- Kivy
